# 광고 소재 성과 분석 툴

## 프로젝트 개요
광고 소재(이미지/영상)의 성과를 AI로 분석하는 툴

## 기술 스택
- **프론트엔드**: React 18 + Vite
- **배포**: Vercel
- **영상 처리**: Cloudinary
- **AI 분석**: Anthropic Claude API

## 분석 항목
- A. 주목 & 후킹
- B. 메시지 & 행동유도
- C. 심리 & 감성설계
- D. 신뢰 & 증거
- E. 브랜드 & 차별성
- F. 매체 & 포맷 최적화
- G. 소재 피로도 & 지속력

## 배포 정보
- **GitHub**: sgon0101/ad-analyzer
- **Vercel**: https://ad-analyzer-sage.vercel.app

## 환경 변수
```
ANTHROPIC_API_KEY
CLOUDINARY_CLOUD_NAME
CLOUDINARY_API_KEY
CLOUDINARY_API_SECRET
```
API 키는 반드시 환경변수로만 사용 (하드코딩 금지)
